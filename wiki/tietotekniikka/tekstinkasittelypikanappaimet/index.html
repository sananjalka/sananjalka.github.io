Pikanäppäimiä tekstin käsittelyyn Windowsilla

Pikanäppäin Selite
Home Siirtää kursorin rivin alkuun.
End Siirtää kursorin rivin loppuun.
Ctrl + Home Siirtää kursorin tiedoston alkuun.
Ctrl + End Siirtää kursorin tiedoston loppuun.
Page Up Siirtää kursorin ruudun korkeuden verran ylöspäin.
Page Down Siirtää kursoria ruudun korkeuden verran alaspäin.


Ctrl + vasen nuolinäppäin Siirtää kursorin kursoria edeltävän tai kursorin sisältävän sanan alkuun.
Ctrl + oikea nuolinäppäin Siirtää kursorin seuraavan sanan alkuun.
Ctrl + askelpalautin Poistaa kursoria edeltävän tai kursorin sisältävän sanan.
Ctrl + Delete Poistaa seuraavan sanan tai kursorin sisältävän sanan loppuun kursorista eteenpäin.
Ctrl + ylempi nuolinäppäin Siirtää kursorin kappaleen alkuun tai yhden rivin ylöspäin (ohjelmakohtaista).
Ctrl + alempi nuolinäppäin Siirtää kursorin kappaleen loppuun tai yhden rivin alaspäin (ohjelmakohtaista).


Shift + vasen tai oikean nuolinäppäin Valitsee kirjaimia yksi kerrallaan.
Shift + ylempi tai alempi nuolinäppäin Valitsee rivejä yksi kerrallaan.
Shift + Ctrl + vasen tai oikea nuolinäppäin Valitsee sanoja yksi kerrallaan.
Shift + Ctrl + ylempi tai alempi nuolinäppäin Valitsee rivejä tai kappaleita (ohjelmakohtaista) yksi kerrallaan.
Shift + Home Valitsee tekstin kursorin ja rivin alun välistä.
Shift + End Valitsee tekstin kursorin ja rivin lopun välistä.
Shift + Ctrl + End Valitsee tekstin kursorin ja tiedoston lopun välistä.
Shift + Ctrl + Home Valitsee tekstin kursorin ja tiedoston alun välistä.
Shift + Page Down Valitsee tekstiä ruudun korkeuden verran kursorin jälkeen.
Shift + Page Up Valitsee tekstiä ruudun korkeuden verran ennen kursoria.
Ctrl + A Valitsee kaiken tekstin.


Ctrl + C Kopioi valittu teksti.
Ctrl + X Leikkaa valittu teksti.
Ctrl + V Liitä teksti kursorin kohdalle.
Ctrl + Z Peru toiminto.
Ctrl + Y Tee uudestaan peruttu toiminto.


Ctrl + F Etsi.
Alt + F Avaa Tiedosto-valikko
Alt + E Avaa Muokkaa-valikko
Alt + V Avaa Näytä-valikko
